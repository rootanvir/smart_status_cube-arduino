## 1.3'' spi oled display pinout

| OLED Pin | ESP8266 (NodeMCU) |
| -------- | ----------------- |
| VCC      | 3.3V              |
| GND      | GND               |
| SCL (D0) | D5 (GPIO14)       |
| SDA (D1) | D7 (GPIO13)       |
| RES      | D4 (GPIO2)        |
| DC       | D3 (GPIO0)        |
| CS       | D8 (GPIO15)       |
